Ethereum 101

Ethereum is a digital platform with which it is possible to create various decentralized applications. It allows you to send cryptocurrency to anyone for a small fee. It also powers apps that anyone can use and no one can take down. It is the world's programmable blockchain.

Ethereum Dapp Programming

A Dapp (decentralized applications) are what distinguishes the Ethereum network from many other blockchain networks. A decentralized application is a software application that runs over a network and is not controlled by a central authority.

Ethereum Smart Contract Programming 101

Smart contracts are a type of Ethereum account. This means that they have a balance and can send transactions over the network. However, they are not controlled by a user, but are implemented on the network and run as scheduled.

Ethereum Smart Contract Programming 201

The Ethereum Smart Contract Programming 201 takes a deep-dive into the Ethereum ecosystem and explores everything from Web3, Truffle, Unit Testing and Dapps to Ethereum Oracle development.
